# EERF Product Roadmap

## Phase 1 — Recovery Foundation (✅)

- Canary CDN+Origin dual-path check
- Auto Failover (Route53 + WAF + SG)
- DNS Validate + Auto Rollback
- Manual Failback
- Cross-Account (STS AssumeRole)

## Phase 2 — Production Readiness (✅)

- Discovery (Organizations dynamic discovery)
- Hourly Diff Engine (change detection)
- Report Generator (Markdown)
- Notification (report + onboarding + deletion alert)
- Approval State (Pending → Approved)
- CloudWatch Dashboard

## Phase 3 — AI Assisted Operations

- AI Risk Analysis
- Onboarding Priority Recommendation
- Explainable Recommendation

## Phase 4 — Enterprise Platform

- DynamoDB Service Registry
- API Gateway
- Web Console
- RBAC
- 100+ Account management
